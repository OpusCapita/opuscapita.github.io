Dashboard

Description

Dashboard is a container component to a pattern for visualizing independent sections of information in an organized manner.

Components

Dashboard is a composition of different components.

Charting

Charting is used for visualizing data e.g. as pie charts.

Grid layout

Grid layouting is used for configurable, draggable and resizable grid layout container for content.

Card

Card is not used yet, but should be considered as a wrapping component for the content in order to have consistent look and feel.
